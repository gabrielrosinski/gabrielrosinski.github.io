Use OpenWeatherMapLib.dll to get functionality of the openweathermap library. Basic usage : Add to your project Reference to OpenWeatherMapLib.dll and type using openweathermap; Basic application: Location newlocation = new Location("London"); IWeatherDataService open_map_weather = WeatherDataServiceFactory.getWeatherDataService(WeatherDataServiceFactory.OPEN_WEATHER_MAP); WeatherData opachki = open_map_weather.getWeatherData(newlocation); opachki.printAllCached(); Creating a Location object you can give it a name of the city or a city id in the construction. By now IWeatherDataService interface have only one function : getWeatherData which return the WeatherData object. WeatherData object will hold information about Temperature of the give City and format of their representation. By now only one Class implement IWeatherDataService, OpenWeatherMan. WeatherDataServiceFactory returns a OpenWeatherMan object if asked to create one.
